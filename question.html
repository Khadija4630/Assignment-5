Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model is that allows developers to access all html elements and modify the elements.

How do you select an element from the DOM?

To select a single element we can use getElementById(),querySelector().To select multiple elements we can use getElementByTagName() ,querySelectorAll() and getElementByClassName().

What is event delegation in the context of the DOM ,and why is it useful?

Event Delegation is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.Event Delegation is a useful pattern that allows you to write cleaner code, and create fewer event listeners for events on many elements with just one event listener.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, you can use methods like setAttribute, getAttribute, and removeAttribute.You can change the styles of an element using the style property.Using the classList property, you can add, remove, or toggle classes.
